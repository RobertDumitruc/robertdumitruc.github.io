ELENOR

A visual brand showcase for a watch concept, built as a single motion piece for social media presentation.

My Notes

The ELENOR project has brought my 3D skills and motion design forward. This project is centred around a single finished video outcome. The idea of this showcase was themed around an icy* environment, eventually revealing the brand name with some special effects alongside it. Working with my friend on this has been a fun exercise and I would love to do something like this again.

*The reasoning behind the icy environment refers to the term "Ice" being used as a blanket term for expensive jewelry.
