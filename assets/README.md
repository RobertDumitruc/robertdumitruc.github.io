# Assets Structure

This folder keeps visual assets organised by page so the project does not get cluttered as more work is added.

Suggested structure:

- `assets/home/`
  Use for images that appear on the landing page, such as homepage thumbnails, profile images, or shared graphics.
- `assets/work/urinetown/`
  Use for images related to the `URINETOWN` detail page.
- `assets/work/short-motion-pieces/`
  Use for stills, posters, or preview frames used by the motion detail page.
- `assets/work/studio-and-university-outcomes/`
  Use for coursework images, process screenshots, and supporting visuals.

Suggested naming:

- `thumbnail.jpg` or `thumbnail.png`
- `hero.jpg`
- `process-01.jpg`
- `process-02.jpg`
- `final-mockup.jpg`

If you add more pages later, create a matching folder inside `assets/work/` using the same naming style as the page file.
